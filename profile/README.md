## 📌 关于本组织

YaoLumen 是一个专注于**业务系统架构**与**电商运营中台**的技术组织。

由于公司战略调整，部分项目已停止公开发布。出于**技术交流**和**个人学习**目的，我们决定将已有项目以**半成品形式**开源，供对相关领域感兴趣的研究者参考。

> **注意**：本组织下的项目为**半成品（架构完整，功能未完结）**，仅供学习研究使用。作者已自用完整版本，请勿直接用于生产环境。

## 💡 技术栈亮点

- **架构设计**：DDD 模块化、TypeORM / Prisma 仓储模式
- **认证体系**：JWT 双 token（Access 2h + Refresh 7d）、RBAC
- **API 设计**：tRPC 类型安全、RESTful、版本化
- **前端**：Next.js 14 App Router、React 18、Tailwind CSS
- **跨系统集成**：WooCommerce 多店对接、AI 评论分析、双向数据同步
- **数据库**：PostgreSQL、Prisma Migrate
- **缓存 / 队列**：Redis
- **容器化**：Docker

## ✨ 值得借鉴的功能

- 🤖 **AI 集成**：智谱 AI 自动分析评论情感与分类
